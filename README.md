## 프로젝트 소개

writeNow는 중앙대학교 예술공학부를 기반으로 운영되는 블로그 스터디 소모임입니다.<br />
테크니컬 라이팅을 통해 개인의 구조적 글쓰기 역량을 강화하고, 블로그 플랫폼을 활용해 지식을 공유하며 협업할 수 있는 기반을 만드는 것을 목표로 활동하고 있습니다.
